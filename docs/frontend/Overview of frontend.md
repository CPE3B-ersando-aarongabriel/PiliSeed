🌱 PiliSeed: Smart Agriculture Advisory System
<br>

**Frontend Documentation**

## 💡Introduction

PiliSeed is a web-based agricultural intelligence platform designed to help farmers make smarter, data-driven decisions. By combining location data, real-time weather information, and soil conditions, PiliSeed provides personalized crop recommendations, risk assessments, and actionable farming insights.

The problem we address is that many farmers rely on generalized advice that does not account for local environmental conditions. This often leads to poor crop selection, reduced yields, and increased vulnerability to climate risks. PiliSeed simplifies decision-making by delivering precise, localized recommendations powered by artificial intelligence.



## 🔮 Features

PiliSeed offers a structured frontend experience that allows users to interact with agricultural data efficiently:

### 🧩 **[1]**. User Interface & Navigation

A clean and intuitive layout that allows users to easily navigate between features, pages, and tools.

***

### 🔐 **[2]**. Authentication Interface

Simple login and signup pages that allow users to securely access personalized features.

***


###  🌾 **[3]**. Farm Input & Management UI

Users can add, edit, and manage farm information through structured and easy-to-use forms.

***


###  🌾 **[4]**. Soil Data Input Interface

Interactive forms allow users to input soil parameters such as pH, moisture, texture, and nutrient levels.

***


###  🌾 **[5]**. Dashboard & Analytics View

A centralized dashboard that displays key insights, including farm overview, weather summaries, and recommendation previews.

***


###  🌾 **[6]**. Crop Recommendation Display

Recommendations are presented clearly with suitability scores, explanations, and filtering options for better decision-making.

***


###  🌾 **[7]**. Weather Visualization

Displays current conditions and forecasts using visual elements such as charts and indicators.

***


###  🌾 **[8]**. Yield & Insights Visualization

Provides visual representations of projected yield, trends, and insights using charts and graphs.

***


###  🌾 **[9]**. Responsive & Accessible Design

The interface adapts across devices and ensures accessibility for a wide range of users.
